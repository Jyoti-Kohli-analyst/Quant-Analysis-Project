# Quant-Analysis-Project
Sector Analysis of Indian IT Companies

Overview :
 This project analyzes the Indian IT sector using historical stock market data from:
 -TCS
 -Infosys
 -Wipro

Objective :
 To identify:
  -Price trends
  -Volatility patterns
  -Trading activity
  -Short-term market behavior

Tools Used:
 -Excel
 -Investing.com Data
 -Data Visualization

Key Insights:

TCS
 -Most stable stock movement
 -Lower volatility
Infosys
 -Higher short-term volatility
 -Faster recoveries
Wipro
 -Highest trading volume
 -Strong investor participation

Outcomes:
Generated a financial research report comparing major Indian IT companies and identifying potential trading and risk insights.
